### 1. Figma 教學整合：元件庫、Design System Tokens 與練習畫面
**功能：** 在 Figma 建立一份對照教材，讓學員比對「設計 vs 程式碼」，並可拿來練習建 design system。
**目的：** Sass / Tailwind 講義與練習原本只有程式碼，缺少視覺化設計對照與可操作的 design token 練習環境。
**實作：**
- Sass 元件庫：Button（primary/success/danger × Default/Hover）、Card、Alert、Token 色板，顏色/間距/圓角綁定 Figma 變數。
- Tailwind 練習畫面稿：名片、登入、響應式、group、深色價目表（含淺/深色）。
- Design System Tokens：Primitives 色階 → Semantic 語義色（別名、含 Light/Dark 兩模式）→ Scale（間距/圓角），加上 Text styles 型別階與 Effect styles 陰影階。
- DS 範例解答：Button / Badge / Input / Card 元件與「設定頁」（Light/Dark 對照），全部綁 Semantic 變數。
- 練習題加入 Figma 參考樣式深連結（node-id），HTML 與 Markdown 版同步。
- Figma 檔案：https://www.figma.com/design/e87BnLqAVr9tTSyCTNP7TX
