# 5. 變數

### **變數命名方式**

Sass / Scss 變數的宣告主要依靠 `$`  

```scss
$a : 10;
$fontSize : 20px;

```

### **數值型態**

- 數值 (Number)：12、100px (可能有或沒有單位)
- 字串 (String)：Microsoft JhengHei (可能有或沒有引號)
- 顏色 (Color)：blue、#4cb5fc、hsl(204, 97%, 64%)
- 列表 (List)：1px 2px 3px 1px   or  1px , 2px ,3px , 1px
- 地圖 (Maps)： (primary: blue, danger: red)
- 布林 (Boolean)：true、false
- 空值 (Null)：null

```scss
//數值型態
$font_size-h1 : 50px; //數值
$font_color: rgb(255, 81, 0); // 顏色
$margin : 10px 20px 30px 40px; //  list
$null : null; //空值
$if : true //判斷式
$bg-color : (
  'blue' : #0059ff,
  'yellow' : #ffd900,
  'green' : #73ff00
); //map
```

### **變數管理技巧**

1. 變數會先宣告，必定要在取用之前進行宣告
2. 把介面共用的參數設定為變數

```scss
//字體
$fontSize :  16px;
$fontWeight: 400;
//顏色
$Body_color: #999;
$body_BgColor : #fff;

```

---

1. 變數命名方式通常以屬性的名稱來命名，要有語義化
2. 區域變數與全域變數(只要在`{}` 裡的通常為**區域變數**)
3. 使用 !default

### 作用域**Scope**

1. 使用 `!default` 改變變數權重

```scss
$d : 200px;
$d : 100px !default ;

.default_scope {
   width: $d;
}

	// ============ 結果 ============ //

```

1. 變數重新賦值 ( 重複的變數宣告就如同重新賦值 )

```scss
$a : 10px;
$a : 20px;

.box {
  width : $a
}
```

1. 變數宣告在區域環境裡 ，如果要在全域取到區域的變數值，要使用 `!global`

```scss
$a : 10px;

.default {
  $b : 400;
  font-size: $a;
  font-weight: $b;
}

 .fail {
   font-weight: $b;// 會失敗，要在$b 使用 !global
}

 .default_2 {
   $a : 30px;
   font-size: $a;
 }
// ============ 結果 ============ //

```

### **字串轉換**

如果你想在字串是使用一個變數，簡單來说，就是使用 `#{}` 来包裹這個變數。

只要在 **url路徑 / class 名稱 / 屬性名稱** 都需要使用**字串轉變數**

```scss
.#{$var} {
    width : 10px
}

.box {
   background-image : url(./images/#{$link}.jpg);
}

.attr {
    #{$width} : 10px;
}

```
