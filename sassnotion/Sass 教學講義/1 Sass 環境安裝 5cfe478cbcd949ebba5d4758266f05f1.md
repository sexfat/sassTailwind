# 1. Sass 環境安裝

## 安裝 [Node.js](https://nodejs.org/en/)

1. 要注意版本，一般我們以LTS 的版本來安裝
2. 安裝時，我們以安裝在系統槽為主

## 安裝 Sass

使用命令提示字元CMD 或是終端機來安裝

```markup
npm install -g sass
```

提醒mac  使用者，**安裝sass必須是管理者權限，可以在安裝時加入**  
`sudo npm install -g sass`

## 檢查是否有安裝成功

```markup
sass --version
```

如果有安裝成功，會有sass 版本
