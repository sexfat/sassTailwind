# 10. Sass Loop 迴圈介紹

迴圈常與資料 list 與 map 一起使用

### **@for 迴圈**

```scss
**@for $i from 1 through 10 {}    // 從起始值 1 到結束值 10 (包含結束值) 
@for $i from 1 to 10 {}    // 從起始值 1 到結束值 10 (不包含結束值)** 
```

```scss
@for $i from 1 through 5 {
    .font-#{$i} {
        font-size: $i * 10px;
    }
}
```

### **@each 迴圈**

```scss
@each $i in < list >{}
@each $i in < map >{}
```

- 圖片路徑

```scss
$images: a1 a2 a3 a4 boxer; //list

@each $img in $images {
    .#{$img}-img {
        background-image:url(./img/#{$img}.jpg);
    }
}
```

- 文字大小

```scss
$font-size: 10, 25, 40; //list型態

@each $size in $font-size {
  .size-#{$size} {
    font-size: $size + px;
  }
}
```

- 使用 map

```scss
$titles : (
    'h1' : 34px,
    'h2' : 30px);

@each $title, $value in $titles {
    #{$title} {
        font-size: $value;
    }
}
```

- 跨瀏覽器

```scss
// prefix 跨瀏覽器 
@mixin prefix($property , $val) {
    @each  $prefix in  '' , -webkit- , -moz- ,-ms-  , -o-{
        #{$prefix}#{$property}:$val;
    }
}

```

```scss
$breakpoint: ( 
    small: 480px, 
    medium: 720px,
    desktop : 988px,
    large: 1366px,
    );

@mixin respond($res) {
  @each $viewpoint,$num in $breakpoint {
      @if $res == $viewpoint {
          @media all and (min-width: $num) {
              @content
          }
      }
  }
};
```
