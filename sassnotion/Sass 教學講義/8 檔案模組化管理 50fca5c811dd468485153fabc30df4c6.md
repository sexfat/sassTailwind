# 8. 檔案模組化管理

管理scss檔案更方便，編譯到同一個css檔案內

使用 @import 導入檔案

```scss
$bgc: #000;

.bg {
   background-color: $bgc;  
}
```

```scss
//導入var.scss 這隻檔案
@import 'var'; 
```

由上面的例子，我們會發現在css資料夾裡，有多一隻 `var.css` 檔案

這不是我們想要的結果，如果有很多個導入的scss檔，表示會產生很多隻css檔

### 使用partials 功能

在檔案前面加入 `_var.scss` ，到時候就會整合成一隻css檔案
