# 4. Vscode Sass 編譯工具

### **Live Sass Compiler**

[https://marketplace.visualstudio.com/items?itemName=ritwickdey.live-sass](https://marketplace.visualstudio.com/items?itemName=ritwickdey.live-sass)

### 套件設定檔修改

1. Open the command palette (either with F1 or Ctrl + Shift + P )
2. Type "open settings"
3. You are presented with two options, choose "Open Settings (JSON)"

![sass.jpg](4%20Vscode%20Sass%20%E7%B7%A8%E8%AD%AF%E5%B7%A5%E5%85%B7/sass.jpg)

- settings.json ( 放入json檔裡 )

```json
//css產出設定
 "liveSassCompile.settings.formats": [
        {
            "format": "expanded", //壓縮成一行"compressed", //壓縮成一行css，正式版本
            "extensionName": ".css",
            "savePath": "/css/" //此為輸出的路徑，可以自行更改
        }
    ],
```
