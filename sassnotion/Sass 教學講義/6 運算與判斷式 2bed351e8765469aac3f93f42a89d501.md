# 6. 運算與判斷式

### 運算

```scss
// 基本數值運算
$width: 100px;
$padding: 20px;
$border: 1px;

.container {
  // 加法
  width: $width + $padding * 2 + $border * 2;  // 結果: 142px
  
  // 減法
  margin-left: $width - 50px;  // 結果: 50px
  
  // 乘法
  line-height: 20px * 1.5;  // 結果: 30px
  
  // 除法 (在新版本中必須使用math.div或括號)
  font-size: (40px / 2);  // 結果: 20px
  
  // 取模 (餘數)
  z-index: 11 % 3;  // 結果: 2
}

```

> 注意：在SASS的最新版本中，直接使用/符號會被視為CSS中的分隔符。
必須使用math.div()函數或使用括號來執行除法運算。
> 

### 單位

```sass
單位運算
SASS會自動處理單位之間的計算：
scss// 相同單位運算
$margin: 20px + 10px;  // 結果: 30px

// 單位與數字相乘
$double-margin: $margin * 2;  // 結果: 60px

// 單位與數字相除
$half-margin: $margin / 2;  // 結果: 15px

// 不同單位 (需要兼容單位)
$padding: 2em + 20px;  // 錯誤：不兼容的單位

// 單位轉換 (使用計算)
$rem-size: 16px;
$element-size: 24px / $rem-size * 1rem;  // 將px轉換為rem
相對單位
計算相對單位時要特別注意兼容性：
scss// 百分比計算
$container-width: 960px;
$sidebar-ratio: 30%;
$sidebar-width: $container-width * $sidebar-ratio;  // 結果: 288px

// em計算 (通常需要上下文)
$base-font-size: 16px;
$h1-size: 2em;
$h1-size-px: $h1-size * $base-font-size;  // 結果: 32px
```

### math模組

在SASS新版中，提供了專門的`math`模組處理數學運算：

```sass
@use "sass:math";

.element {
  // 除法
  width: math.div(100px, 2);  // 結果: 50px
  
  // 四捨五入
  padding: math.round(7.6px);  // 結果: 8px
  
  // 四捨五入
  font-size: math.round(14.4px);  // 結果: 14px
  
  // 向上進位
  top: math.ceil(14.1px);  // 結果: 15px
  
  // 向下取整
  bottom: math.floor(14.9px);  // 結果: 14px
  
  // 生成隨機數 (0-1之間)
  opacity: math.random();
```

### 顏色函數

SASS提供了豐富的顏色函數：

```sass
@use "sass:color";

$base-color: #3498db;

// 調亮顏色
background-color: lighten($base-color, 20%);  // 提高20%亮度
background-color: color.adjust($base-color, $lightness: 20%);  // 同上
  
// 調暗顏色
color: darken($base-color, 10%);  // 降低10%亮度
color: color.adjust($base-color, $lightness: -10%);  // 同上

// 灰度化
background-color: grayscale($base-color);  // 轉換為灰度
  
// 反轉顏色
border-color: invert($base-color);  // 反轉顏色
```

## 字串運算

### 字串連接

SASS中的字串可以使用`+`運算符連接：

```scss

$asset-path: "assets/";
$image-name: "logo.png";

.logo {
// 連接字串
  background-image: url($asset-path + $image-name);// 結果: url(assets/logo.png)

// 連接字串和數值
  content: "Chapter " + 5;// 結果: "Chapter 5"

// 多個字串連接
  font-family: "Helvetica" + ", " + "Arial" + ", " + "sans-serif";
}

```

### 字串插值

使用`#{}`進行字串插值，將變數或表達式嵌入到字串中：

```scss

$name: "main";
$attribute: "border";
$direction: "top";
$base-font-size: 16px;

.element {
// 在選擇器中使用插值
  &-#{$name} {
// 在屬性名中使用插值
    #{$attribute}-#{$direction}: 1px solid black;

// 在屬性值中使用插值
    margin-#{$direction}: 20px;

// 在計算中使用插值
    font-size: #{$base-font-size * 1.5}px;

// 在媒體查詢中使用插值
    @media (min-width: #{600 + 100}px) {
      display: block;
    }
  }
}

```

### 判斷式

```scss
$aa : false !default;
$aa :true;
.w {
  @if $aa
  {
     color: #000
  } @else {
     color: #fff
  }
}
```

- 判斷式條件

and：兩個運算式都是 true 時才會回傳 true
or：兩個運算式有任一個是 true 時就會回傳 true
not 反轉運算式結果

```scss

@if $a == #000 and $a == black {
   color : #000;
}@else{
   color: #a;
}
```

- 使用判斷式來控制 滿版 與 固定版型 的設定

```scss
$num :1366px;

.container {
 @if $num == full{
    width:  100%;
   
 }@else {
     width: $num;
      margin: 0 auto;
 }
```

- @else if

```scss
@mixin text($fontstyle){
    color : #000;
    @if $fontstyle == heavy {
       font-size: 40px;
       font-weight: bold;    
    }@else if $fontstyle == medium {
       font-size: 28px;
       font-weight: 400;
    
    }@else if $fontstyle == light {
       font-size: 16px;
       font-weight: 100;
    } 

}

.nav {
  @include text(light);
}

```

- 結合@mixin 判斷使用 border-radius
