# 7. 巢狀寫法Nesting

讓 css 可讀性更高

### **使用nesting的黃金規則：**

1. nesting 建議不要超過三個層級
2. 確保輸出的CSS簡潔、可重用
3. 使用nesting是很有意義的，而不是默認選項
4. 結合BEM 

```scss
nav {
  ul {
    margin: 0;
    padding: 0;
    list-style: none;
    
    li { 
      display: inline-block; 
       }
  }
}
```

### 偽元素

- css

```scss
/*scss如何寫*/

a:hover { color:#333}
p::after {content : ''}
```

- scss

```scss
a {
  &:hover {color: #333}
}

p {
  &::after { content : ''}
}
```

### 延伸寫法

樣式的多樣性寫法，語法層級更易讀

```scss
.btn {
  
  &--dark {
    background-color: #000;
  }
  &--light {
     background-color: #fff;
     
  }

}
```
