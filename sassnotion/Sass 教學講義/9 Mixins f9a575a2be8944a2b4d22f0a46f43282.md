# 9. Mixins

使用@mixin  &  @include 來定義重複樣式

### 宣告 mixin ，把常用的樣式給定義出來

```scss

@mixin name(){  
  margin-top : 100px
}

@include name();  
```

### 置入變數

```scss
@mixin layout($theme) {
    @if $theme==full {
        width: 100%
    }

    @else {
        width: $theme + px;
        margin: auto;
    }

}

// 使用 @include layout(full)
```

### 變數預設值

```scss
@mixin space($value : 100px){
    margin-top: $value;
}

// 會帶入預設值100px	
// @include space();
```

### 變數空值

```scss
//設定圓角空值
@mixin btn($w , $color , $border_radius: null) {
    padding:10px $w;
    background-color: $color;
    color:#fff;
    text-align: center;
    border-radius:$border_radius;
}

.btn-default {
   @include btn(30px,blue)
}
```

### 呼叫另一個mixin

```scss
@mixin margin-auto() {
    margin-left: auto;
    margin-right: auto;
}

@mixin rect($width , $height: $width) {
    width: $width;
    height: $height;
    //可以呼叫另一個函式
    @include  margin-auto()
    
}
```

### 動態加入樣式跟屬性

@content  裡是放置新增的屬性

```scss
@mixin m() {
    @media screen and (max-width: 576px) {
        @content;
    }
}

//使用
@include m(){
    //手機版樣式
}
```
