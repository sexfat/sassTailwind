# Sass & Tailwind CSS 排課表（總時數 15 小時）

> 建議切成 **8 堂課**：Sass 8 小時（第 1–4 堂）＋ Tailwind 7 小時（第 5–8 堂）。
> 每堂課皆採「講解 → 帶做範例 → 學員練習 → 講評」的節奏。時間為含中場休息的抓法，可依班級進度彈性調整。

## 第一部分：Sass（8 小時）

### 第 1 堂　Sass 入門：環境、變數、巢狀（2.0h）
| 時間 | 內容 |
|------|------|
| 20m | Ch.1 什麼是 Sass、為什麼用、SCSS vs Sass 語法 |
| 25m | Ch.2 安裝與編譯：Node.js(LTS)、`npm install -g sass`（Mac 需 sudo）、`--watch`、Live Sass Compiler 設定 |
| 35m | Ch.3 變數：7 種資料型態、作用域、`!default`／`!global`、`#{}` 插值、Sass 變數 vs CSS 變數 |
| 25m | Ch.4 巢狀與 `&` 父選擇器、BEM、巢狀屬性、「不超過 3 層」原則 |
| 15m | 帶做 `examples/01-variables.scss`、`02-nesting.scss` |
| 20m | **學員練習 1–3** + 講評 |

### 第 2 堂　Sass 模組化與重用（2.0h）
| 時間 | 內容 |
|------|------|
| 30m | Ch.5 Partials、`@use`／`@forward`、命名空間、`as *`（強調取代舊的 `@import`）|
| 35m | Ch.6 Mixin：`@mixin`／`@include`、參數與預設值、`@content`、響應式斷點 mixin |
| 20m | Ch.7 `@extend` 與 `%placeholder`、與 mixin 的取捨 |
| 15m | 帶做 `examples/03-mixin.scss`、`04-extend.scss`、`06-modules/` |
| 20m | 練習：把上一堂成果拆成 partials + 用 `@use` 組回來 |

### 第 3 堂　Sass 函式與運算（2.0h）
| 時間 | 內容 |
|------|------|
| 30m | Ch.8 `@function`／`@return`、內建模組（math／color／string／list／map）|
| 30m | Ch.9 運算子、除法陷阱與 `math.div()`、字串串接與插值 |
| 20m | 顏色函式：`color.adjust`／`color.scale`／`rgba` 實作示範 |
| 20m | 帶做 `examples/05-function-loop.scss` |
| 20m | **學員練習 4–6** + 講評 |

### 第 4 堂　Sass 控制流程與實戰（2.0h）
| 時間 | 內容 |
|------|------|

## 排課備註
- **時數彈性**：每堂預留約 10–15% 緩衝，實際講解常會超時，練習時間可視情況壓縮或延到下堂開頭。
- **課前準備**：第 1 堂前請學員裝好 VS Code + Live Sass Compiler；第 5 堂僅需瀏覽器（CDN）。
- **若總時數需壓縮**：Sass Ch.9 運算、Tailwind Ch.10 `@apply` 可改為自習；`@while`、v3 config 可略講。
- **若時數有餘裕**：可在第 4／第 8 堂加做一個「Sass→Tailwind 同題目兩種寫法」對照，加深兩者定位差異。
