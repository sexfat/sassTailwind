Tailwind CSS 練習題

每題都有一個 起始檔 供你動手，解答/ 資料夾有完整答案。用瀏覽器（或 VS Code Live Server）打開 HTML 即可即時看效果。所有練習皆用 Play CDN，不需安裝。回 Tailwind 講義。

怎麼用 AI 幫你學（先看這裡）

每題下方都有一個「AI 協助」框，附上可直接複製的 prompt。用 AI 學 Tailwind 的 4 個要領：

  • 指定版本：說「請用 Tailwind CSS v4、Play CDN」，避免 AI 給到過時的 config 寫法。
  • 用「需求」描述，讓 AI 給你 class：例如「元素要置中、白底、圓角、陰影，該用哪些 utility class？」
  • 先要提示再要完整碼：加「先列出會用到的 class 清單並解釋，再讓我自己組」。
  • 貼 HTML 請它優化：把你的 HTML 貼上，問「怎麼改成 RWD？」「這串 class 有沒有更簡潔的寫法？」

練習 1：個人名片

Figma 參考樣式：開啟「TW 01 名片」畫面

做一張置中的名片：

  • 頁面背景灰色（bg-gray-100），名片用白底、圓角、陰影、內距。
  • 名片寬度最大 max-w-sm，水平置中（mx-auto）。
  • 內含：姓名（大字粗體）、職稱（灰字小字）、一個藍色按鈕「聯絡我」。
  • 按鈕 hover 時變深藍。
AI 協助 · 可以這樣問
我在學 Tailwind CSS v4（Play CDN）。要做一張置中名片：
頁面 bg-gray-100、名片白底＋圓角＋陰影＋內距、max-w-sm 水平置中，
內含姓名(大字粗體)、職稱(灰字小字)、藍色按鈕「聯絡我」，按鈕 hover 變深藍。
請先「列出會用到的 utility class 並解釋」，不要直接給完整 HTML。

追問：「水平置中除了 mx-auto 還有哪些做法？差在哪？」

練習 2：水平垂直置中的登入框

Figma 參考樣式：開啟「TW 02 登入」畫面

  • 整個畫面高度滿版（min-h-screen），登入框在正中央。
  • 框內：標題、一個 input（有 border、圓角、focus:ring）、一個滿寬按鈕。
AI 協助 · 可以這樣問
Tailwind v4：我要一個「整頁高度、登入框在正中央」的版面。
請教我用 min-h-screen + flex + items-center + justify-center 做垂直水平置中，
框內有標題、input(border+圓角+focus:ring)、滿寬按鈕(w-full)。
先解釋這幾個置中相關 class 各自負責什麼，再讓我組。

focus:ring 沒反應時，貼上 input 那行問：「為什麼點下去沒有外框光暈？」

練習 3：響應式導覽列 + 卡片牆

Figma 參考樣式：開啟「TW 03 響應式」畫面

  • 頂部導覽列：左邊 LOGO、右邊三個連結，兩端對齊、垂直置中。
  • 下方卡片牆：手機 1 欄、平板（md）2 欄、桌機（lg）3 欄，gap-6。
  • 每張卡片 hover 時陰影加深。
AI 協助 · 可以這樣問
Tailwind v4 響應式：請教我 mobile-first 斷點觀念。
我要一個導覽列(左 LOGO、右三連結，兩端對齊垂直置中)，
下面卡片牆用 grid：手機 1 欄、md 2 欄、lg 3 欄、gap-6，卡片 hover 陰影加深。
請先解釋「沒有前綴 = 手機，md:/lg: 往上加」，再列出 grid 相關 class。

追問：「grid-cols-1 md:grid-cols-2 lg:grid-cols-3 為什麼要從小寫到大？反過來寫會怎樣？」

練習 4：group-hover 互動卡片

Figma 參考樣式：開啟「TW 04 group」畫面

做一張卡片，滑鼠移到整張卡時：

  • 卡片陰影加深
  • 標題文字變色
  • 卡片內的「箭頭 →」向右移動一點（提示：group-hover:translate-x-1）
AI 協助 · 可以這樣問
Tailwind v4：請教我 group / group-hover 的用法。
我要一張卡片，滑鼠移到「整張卡」時：陰影加深、標題變色、內部箭頭 → 向右移(translate-x-1)。
請先解釋：父層加 group、子層用 group-hover: 前綴的運作原理，
以及為什麼是 hover 父層而不是子層自己。再讓我試寫。

進階：問「group 和 peer 差在哪？什麼情境該用哪個？」

練習 5：可切換的深色模式價目表

Figma 參考樣式：開啟「TW 05 價目表」畫面（含淺/深色）

  • 用 @theme 自訂一個品牌色。
  • 做三欄式價目表（Free / Pro / Team），Pro 方案要用品牌色外框「突顯」。
  • 右上角一個按鈕可切換深色模式，整頁在深/淺色都要好看（善用 dark:）。
  • 桌機三欄、手機一欄（RWD）。
AI 協助 · 可以這樣問（挑戰題建議分段問）
我要用 Tailwind v4（Play CDN）做「可切換深色模式的三欄價目表」，請分步驟帶我：
步驟1：怎麼在 CDN 頁面用 @theme 自訂一個品牌色？
步驟2：三欄價目表(Free/Pro/Team)，桌機 3 欄手機 1 欄，Pro 用品牌色外框突顯。
步驟3：右上角按鈕切換深色模式(class 策略)，並用 dark: 讓深/淺色都好看。
每步先解釋再給範例，讓我逐步完成。

深色模式切換牽涉一小段 JS（toggle html 的 class）。可單獨問：「請給我切換 dark class 的最小 JS。」
